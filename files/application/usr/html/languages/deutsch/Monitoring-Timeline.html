Monitoring Timeline

Einführung

Diese Modul ermöglicht einen schnellen Vergleich der eingegangenen Syslog/SNMP-Nachrichten innerhalb eines definierten Zeitraums. Dies kann unter anderem auch hilfreich sein, um Unregelmässigkeiten festzustellen.

Benutzung

  1. Als erstes soll der Zeitraum für die Abfrage ausgewählt werden. Dazu im Feld [Erstmals] über das Kalender Icon das Startdatum des Zeitraums wählen und auf dieselbe Weise für das Feld [Letztmals] das Enddatum setzen.
  2. Anschliessend im Feld [Stufe] über das Drop Down Menü bestimmen, in welche Zeiteinheit die eingegangenen Nachrichten zusammengefasst werden sollen (Stunden, Tage (24 Stunden), Wochen oder Monate).
  3. Optional kann man durch das Setzen des Häkchens im Feld [Alternativ] angeben, dass die eingegangen Nachrichten getrennt nach Level angezeigt werden sollen.
  4. Abschliessend die Abfrage über den Button "Zeigen" starten.
Weitere Links nach einer Abfrage:
  • Klickt man im Feld [Zeit] auf einen Zeitpunkt, dann wird man zum Modul {Monitoring Meldungen} weitergeleitet. Dort werden dann die in diesem Zeitraum eingegangenen Meldungen angezeigt.
  • Hat man im Feld [Alternativ] das Häkchen gesetzt, können die Symbole der unterschiedlichen Nachrichten-Levels im Feld [Events] ebenfalls angeklickt werden. Klickt man auf ein solches Symbol, werden im Modul {Monitoring Meldungen} innerhalb dieses Zeitraumes (Stunde, Tag, Woche oder Monat) nur noch Nachrichten des jeweiligen Levels angezeigt.
