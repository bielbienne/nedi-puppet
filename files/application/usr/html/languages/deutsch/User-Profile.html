Benutzer Profil

Einführung

Diesea Modul erscheint als Startseite, sobald sich ein Benutzer am NeDi anmeldet hat. In diesem Modul kann jeder Benutzer individuell sein Profil einrichten. Weiter dient es auch dazu, sich über Meldungen des NeDi-Administrators zu informieren.

Benutzung

Alle Anpassungen in diesem Modul müssen mit dem Button "Aktualisierung" bestätigt werden. Wird die Sprache oder das Darstellungsschema geändert, muss der Button "Aktualisierung" ein zweites Mal angeklickt werden, damit die Änderung sofort ersichtlich ist.
  • Über das Feld [Passwort] kann ein neues Passwort eingegeben und anschliessend im Feld [Quittierung] bestätigt werden. Abschliessend mit dem Button "Aktualisierung" das neue Passwort übernehmen.
  • Im Feld Email kann die Email-Adresse eingetragen werden. Benutzer mit eingetragener Email-Adresse werden durch NeDi über Netzwerkausfälle per Email informiert, sofern diese Funktion im Modul {Monitoring Setup} für die Geräte aktiviert worden ist und der Benutzer zusätzlich der Monitoring Gruppe angehört.
  • Im Feld [Phone] kann die Telefonnummer des Benutzers eingetragen werden. Benutzer mit eingetragener Telefonnummer werden durch NeDi über Netzwerkausfälle via SMS informiert, sofern diese Funktion im Modul {Monitoring Setup} für die Geräte aktiviert worden ist und der Benutzer zusätzlich der Monitoring Gruppe angehört.
  • Im Feld [Kommentar] kann der Benutzer die Hauptaufgabe eintragen, welche er mit NeDi wahrnimmt z.B. First Level, NeDi Administrator etc.
  • Das obere Drop Down Menü linkerhand des Buttons "Aktualisierung" erlaubt dem Benutzer, die bevorzugte Sprache für sein Profil zu wählen. Über das zweite Drop Down Menü unterhalb kann der Benutzer sein bevorzugtes Darstellungsschema für NeDi wählen.

Als Mitglied der NeDi-Administratorengruppe kann ein Benutzer im unteren Teil dieses Moduls über das Icon eine Nachricht kreieren, welche für alle NeDi-Benutzer ersichtlich ist. Dies kann z.B. hilfreich sein, um über aktuelle Netzwerkarbeiten zu informieren.
