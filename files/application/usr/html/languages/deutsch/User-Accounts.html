Benutzer Accounts

Einführung

Dieses Modul ist den NeDi-Administratoren vorbehalten. Ein Administrator kann hier den NeDi-Benutzern Gruppen zuweisen, die Passwörter zurücksetzen oder auch vorhandene Benutzer löschen.

Benutzung

Im Feld [Benutzer] kann über das Textfeld ein Benutzername eingeben und anschliessend der neue Benutzer über den Button "Hinzufügen" erstellt werden. Über Drop Down Menü im Feld [Gruppen] können die bereits vorhanden Benutzer nach Gruppenzugehörigkeit gefiltert aufgelistet werden.

Im Feld [Zeit] ist für jeden Benutzer das Erstellungsdatum hinterlegt. Im Feld [Letztmals] ist ersichtlich, wann sich der Benutzer letztmals beim NeDi angemeldet hat.

  • Über dieses Icon im Feld [Action] kann bei einem Benutzer das Passwort zurück gesetzt werden.
  • Mit diesem Icon im Feld [Action] wird der jeweilige NeDi-Benutzer aus der Datenbank gelöscht.
Für das Zuweisen der Gruppen können im Feld [Gruppen] die nachfolgenden Symbole verwendet werden. Die Gruppen werden einem Benutzer durch Anklicken zugewiesen respektive durch erneutes Anklicken wieder entzogen:

  • Gruppe -> Anderes (oth)
  • Gruppe -> Manager (mgr)
  • Gruppe -> Monitoring (mon)
  • Gruppe -> Helpdesk (dsk)
  • Gruppe -> Network (net)
  • Gruppe -> Administratoren (adm)
Welcher Gruppe welche Module zugeordnet sind, kann in der Datei "nedi.conf" eingesehen und bei Bedarf angepasst werden. Bei Anpassungen in der Datei "nedi.conf" ist darauf zu achten, dass bei Abständen immer mit dem Tabulator und nicht mit Leerzeichen gearbeitet werden darf!
