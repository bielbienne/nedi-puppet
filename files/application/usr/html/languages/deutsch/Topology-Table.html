Topologie Tabelle

Einführung

Die Voraussetzung für dieses Modul sind korrekt konfigurierte SNMP Standort-Angaben auf den Netzwerkgeräten. Diese Standort-Informationen werden in diesem Modul tabellarisch abgebildet.

Benutzung

Die Topologie ist hierarchisch aufgebaut.
  • Durch das Anklicken eines Gebäudesymbols erreicht man die nächst höhere Ebene. Auf der untersten Ebene werden die Netzwerkgeräte des jeweiligen Standortes angezeigt.
  • Klickt man z.B. anstelle eines Gebäudesymbols den Namen des Standorts an, werden direkt alle zugehörigen Netzwerkkomponenten im Modul {Geräte Liste} aufgelistet.
  • Klickt man auf das Symbol eines Netzwerkgerätes, werden detaillierte Informationen zu diesem Gerät im Modul {Geräte Status} angezeigt.
  • Klickt man anstelle eines Gerätesymbols auf die zugehörige IP Adresse, wird eine Telnet-Verbindung zu diesem Gerät aufgebaut.
