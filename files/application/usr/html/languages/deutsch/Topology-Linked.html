Link Editor

Einführung

Werden durch Discovery-Protokolle wie CDP oder LLDP nicht alle Verbindungen korrekt abgebildet, können in diesem Modul diese Verbindungen manuell erstellt oder angepasst werden.

Benutzung

  • Im Feld [Gerät] das gewünschte Netzwerkgerät auswählen. Es wird nun ein weiteres Feld eingeblendet, in welchem die gewünschte Schnittstelle ausgewählt werden kann. Ist die Schnittstelle ausgewählt worden, kann in den nun zusätzlich angezeigten Feldern die VLAN-ID, die Duplex-Einstellung und die Bandbreite festgelegt werden.
  • Dasselbe Vorgehen nun für das Feld [Nachbar] wiederholen. Anschliessend mit dem Button "Hinzufügen" die Anpassungen übernehmen.

Wird ein Gerät im Feld [Gerät] ausgewählt, werden automatisch alle bereits vorhandenen Verbindungen angezeigt. Klickt man in der dargestellten Liste der Verbindungen auf den Namen eines Netzwerkgerätes, werden im Modul {Geräte Status} detaillierte Informationen zu diesem Gerät angezeigt. Eine vorhandene Verbindung kann über das Symbol aus der Datenbank gelöscht werden.

Um die in der Datenbank vorhanden Links anzusehen, kann via dem Drop Down Menü ein entsprechender Eintrag z.B. CDP ausgewählt werden. Es werden dann von allen Netzwerkgeräten entsprechend der Auswahl die zugehörigen Verbindungen aufgelistet.
