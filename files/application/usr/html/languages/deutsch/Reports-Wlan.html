Wlan Access Points

Einführung

Dieses Modul unterstützt das Entdecken von (unerwünschten) Wireless-Access-Points über das LAN-Netzwerk. Alle Endgeräte werden mit einer Liste von WLAN-Access-Point MAC-Adressen verglichen. Stimmt ein solcher Vegleich überein und es werden an einer Netzwerkschnittstelle mehrere Endgeräte gefunden, dann wird dies in diesem Modul angezeigt.

Benutzung

  • Wenn auf einer Schnittstelle nur eine MAC-Adresse gefunden wird, kann, um die Einträge anzuzeigen, im Feld [Alternativ] das Häkchen gesetzt werden.
