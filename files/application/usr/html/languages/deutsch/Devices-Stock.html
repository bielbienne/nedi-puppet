Gerätebewirtschaftung

Einführung

Dieses Modul ermöglicht, unbenutzte Hardware zu verwalten. Wird die Seriennummer eines Moduls oder eines Gerätes im Netzwerk erkannt, wird dieses automatisch aus dem Lager entfernt. Zusätzlich wird eine Mitteilung im Modul {Monitoring Meldungen} ausgegeben.

Benutzung

Um den Typ und die Seriennummer maschinell einzulesen, kann ein Barcode Scanner verwendet werden. Dieser sollte bei erfolgreichem Scannen ein "tab" senden. Eine manuelle Eingabe dieser Daten ist ebenfalls möglich.

Das Gerät oder Module können durch einen Klick auf das Symbol vom Lager entfernt werden. Soll z.B. bei eingescannten Geräten der Standort angepasst werden, muss dieses gelöscht und manuell neu erstellt werden.
