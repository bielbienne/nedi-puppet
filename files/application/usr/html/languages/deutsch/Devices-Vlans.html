Vlan Liste

Einführung

Dieses Modul listet die Vlan-IDs der Geräte auf. Weiter kann angezeigt werden, wie viele Geräte sich in einem Vlan befinden.

Benutzung

Mit dem Button "Zeigen" können die Vlans der Geräte aufgelistet werden. Im Feld Anzeigen können darzustellende Werte hinzugefügt oder weggenommen werden (Ctrl + Mausklick). Es stehen folgende unterschiedliche Filtermethoden zur Verfügung, um die Suche anhand von bestimmten Kriterien einzugrenzen.
  1. Innerhalb der Bedingung A kann über das Drop Down Menü der gewünschte Abfrage-Wert ausgewählt werden z.B. <Duplex> oder <Device>. Diese Abfrage kann durch das zweite Drop Down Menü der Bedingung A mit den Operatoren und durch Eingabe von Text weiter eingeschränkt werden.
  2. Die Abfrage der Bedingung A kann mit einer Abfrage der Bedingung B über das Drop Down Menü im Feld Verknüpfen kombiniert werden.

Details

  • Um mit diesem Modul zu arbeiten ist das Verstehen von Regular Expressions (Abk. RegExp oder Regex) hilfreich.
  • Abfragen können als Favoriten im Browser hinterlegt werden.
  • Am Ende der Tabellen steht immer die resultierende SQL-Abfrage. Diese kann z.B. im Modul "Other -> Export" weiter verwendet werden.
  • Durch das Klicken auf einen Gerätenamen gelangt man direkt zur Statusübersicht des jeweiligen Gerätes. Klickt man in der Spalte Population (Bevölkerung) auf die Anzahl der Geräte im jeweiligen Vlan, dann wird man direkt zum Modul "Nodes -> List" weitergeleitet, welches detailliert die Geräte in diesem Vlan anzeigt.
Links in der Liste der Geräte und VLANs:
  • Der Gerätename in der Spalte [Device] ist mit gesetztem Filter für das jeweilige Gerät mit dem Modul {Device Status} verlinkt.
  • Die angegeben Anzahl der Nodes in der Spalte [Population] ist mit dem Modul {Nodes Liste} verlinkt und zeigt die vorhanden Nodes im jeweiligen VLAN.
