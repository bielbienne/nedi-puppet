Node Reports

Introduction

Reports focused around the computers connected to your network.

Usage

  • Filter on certain values, if desired (won't be taken into account for links to other modules).
  • Select (hold down CTRL for multiple) reports you wish to look at.
  • Adjust the limit to see more or less results.
  • Check 'Alternative' to sort keys rather than values. You see the what's sorted on, on the bottom of each table.

Reports

  1. Node statistics at a glance.
  2. Show duplicate node names (e.g. having a Wlan and Ethernet connection) or MAC addresses
  3. Interface statistics to detect hubs or virtual machines on and NeDi's internal metric statistics.
  4. Show top node OS and type statistics if nodes are fingerprinted with nmap
  5. IP and IF changes multiplied yield NeDi's nomad factor, an indicator for those who seem to travel a lot.
  6. Node update hisory can reveal major changes or problems in your network. Each coloumn is limited individually (use filter to narrow down the timeframe)
  7. Empty vlans can be identified and eventually removed, if not needed on particular devices.
