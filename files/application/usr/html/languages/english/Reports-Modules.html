Module Reports

Introduction

Need to know how many modules of a kind you've got? This report also helps, if you need to generate a HW inventory for support contracts based on serial numbers etc.

Usage

  • Filter on certain values, if desired (won't be taken into account for links to other modules).
  • Select (hold down CTRL for multiple) reports you wish to look at.
  • Adjust the limit to see more or less results.
  • Check 'Alternative' to sort by model (in Summary) or device type (in Inventory) rather than values or device name.

Reports

  1. Distribution presents an overview of which modules are installed in which devices.
  2. Inventory lets you generate a detailed list of devices and their individual modules.
  3. Print supplies and virtual machines are specific reports to reflect these items, which are stored in the DB as modules.
