Devices Stock

Introduction

This module provides the life-cycle management for your devices. If an in-stock module or device's serial number is discovered on the network, its status will be set to active, a comment will be added and an event created (if configured).

Usage

  • Simply use a bar code scanner (should send a "tab" upon successful reading) to scan type and serial number or enter them manually. Specify location, condition, source/provider and warranty, if you care. If the latter one is closer than a month it'll be highlited with the "warning" color or with the "critical" color if expired. A zero with no highligting is shown if no warranty is set.
  • Select a link from any of the summaries to list individual items.
  • If you list by a property (e.g. location), the appropriate field on the top is populated as well for easier batch additions.
  • You can edit items listed by clicking on its status icon. The current list will stay. Note that the focus will move to the location field (SN cannot be edited anyway, since it's the reference). You can either update or delete an item now.
  • You can export a list as XLS by clicking on the the list icon in the header (only appears when a list is actually shown).
