Monitoring Timeline

Introduction

Examine events over time to disclose abnormal behaviour in the past.

Usage

  • It's easier, when you get to this module by clicking on a top-graph
  • Use the filter to narrow down the events
  • Select start and end point and the granularity for your analysis
  • Group the events by level, source or class and select appropriate size of the bars
