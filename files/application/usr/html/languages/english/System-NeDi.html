Run NeDi

Introduction

Execute nedi.pl from the GUI.

Usage

The module is divided in to the following parts:
  1. The default is --help, which reveals options and the output legend.
  2. Definitions shows all available .def files.
  3. Discover will actually find devices (a limit of 100 devices is added for the WebGUI).
  4. Scan IP to search open ports on an ip.
  5. Init is the most dangerous one here and will only proceed with the right DB admin credentials and after confirmation. It will delete and recreate the whole database, but not remove any config files or RRDs.

Details

  • It's not intended for a full blown discovery, but helps to quickly update a device or backup its confguration.
  • Don't worry, if nothing happens right away, give it some time to complete.
