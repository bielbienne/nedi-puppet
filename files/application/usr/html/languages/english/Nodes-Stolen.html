Nodes Stolen

Introduction

The idea behind this module is to primarily mark stolen nodes (notebooks) and notify the help desk if it reappears on the network. It can also be used to rediscover banned and rogue WLAN access points (which where detected by Reports-WLAN for example).

Usage

Click on in the Nodes-Status toolbar or add manually in this module, if you don't have an entry in the nodes table.
