Manage System Files

Introduction

This modules provides 4 major features:
  1. Edit/View system, device configuration and nedi log files.
  2. Import SQL data or update NeDi files.
  3. Manage html/log, map, foto and tftpboot files.
  4. Delete "broken" RRDs which haven't been updated for more days than retire is set to in nedi.conf.

Editor/Viewer

  • Simply choose the file you want to edit and click save, when you're done.
  • You can only edit files, which are writable by the webserver.
  • A confirmation dialog appears, when selecting another file after you've changed the current one.

Import/Update

  • Upload a .sql.gz (packed) file which will replace the DB data. You can restore dumps created with System-Export for example
  • Upload an archive with alternative user icons (usr/0-99.jpg) or device panels (panel/devtype.jpg).
